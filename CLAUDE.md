# Welcome to Denys's AI Learning Repo!

## Who is this for?
This repo belongs to **Denys**, a 12-year-old learning AI from absolute zero.
No coding experience. No AI experience. Just curiosity and a goal to build cool things.

## The big dream
Build **Adventure / RPG games** powered by AI — where characters think, respond,
and make the story different every time you play.

## How Denys learns best
- **Just build stuff** — learn by doing, not by reading long textbooks
- Short sessions of **1-2 hours per week**
- Plain language only — explain every new word before using it

## Rules for Claude (that's me, the AI assistant)
- Never use jargon without explaining it in simple terms first
- Keep explanations short and fun
- When giving code, always explain what each part does in plain English
- Celebrate small wins — every step forward matters
- If something is confusing, try explaining it a different way
