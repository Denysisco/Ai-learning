# Denys's AI Learning Roadmap

> Goal: Learn AI from zero and build an Adventure RPG game powered by AI.

---

## How to read this file
- [ ] means "not done yet"
- [x] means "done — nice work!"
- Each phase has lessons. Each lesson is a small project or skill.

---

## Phase 1 — Talk to AI Like a Pro
> Learn how to give AI good instructions. This skill is called prompting.
> A prompt is just what you type to an AI to tell it what to do.

- [x] Lesson 1.1 — What is a prompt? Write your first one.
- [ ] Lesson 1.2 — Learn what makes a prompt good vs bad (with examples)
- [ ] Lesson 1.3 — Use prompts to generate a short story character for your future game
- [ ] Lesson 1.4 — Make the AI play a simple "yes or no" adventure with you using only prompts

---

## Phase 2 — Build Your First Text Adventure Game (No Coding!)
> A text adventure is a game where you read a story and type your choices.
> Think of it like a "Choose Your Own Adventure" book — but the AI writes it live.

- [ ] Lesson 2.1 — Design your game world (characters, places, a quest)
- [ ] Lesson 2.2 — Write a prompt that makes AI run your game
- [ ] Lesson 2.3 — Playtest it and improve your prompts
- [ ] Lesson 2.4 — Share your game with someone and watch them play

---

## Phase 3 — Learn Python Through Games
> Python is a coding language. It looks like normal English sentences, which makes
> it great for beginners. We'll use it to build game pieces, not boring exercises.

- [ ] Lesson 3.1 — What is Python? Run your very first line of code.
- [ ] Lesson 3.2 — Variables: teach the computer to remember things (like a player's name)
- [ ] Lesson 3.3 — If/else: make the game react to choices
- [ ] Lesson 3.4 — Functions: give a group of steps a name so you can reuse them
- [ ] Lesson 3.5 — Build a mini text adventure in Python (no AI yet)

---

## Phase 4 — Add AI to Your Game
> Now we connect Python to a real AI brain using something called an API.
> API = a door between your game and the AI. Your game knocks, AI answers.

- [ ] Lesson 4.1 — What is an API? (Simple explanation + first test)
- [ ] Lesson 4.2 — Connect Python to Claude AI (Anthropic API)
- [ ] Lesson 4.3 — Make a game character that responds differently every time
- [ ] Lesson 4.4 — Build a full mini RPG with an AI-powered villain
- [ ] Lesson 4.5 — Share your finished game!

---

## Bonus Goals (unlock after Phase 4)
- [ ] Add a map to your game
- [ ] Give your AI character a memory (it remembers what you did earlier)
- [ ] Build a website so others can play your game online
- [ ] Create a second, bigger game from scratch
